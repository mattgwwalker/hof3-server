A test page

This page tests the basic hosting capabilities of the server.

The EventSource data-stream shows the server's ability to query the PLC (but is only displayed on-the-fly by Chrome). If that's not working then we can ask for an EventSource based on the server's system time, this removes communications with the PLC as a potential problem.

We can also view a JSON-formatted output of both the system time and the PLC's time.
